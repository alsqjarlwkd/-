# 그림판 설명

1.프로그래스바를 사용하여 마우스로 panint가 될때 두께 조정 기능적용

2.하단에 색상 클릭시 paint 색상 변경

3.fill 버튼 클릭시 캔버스 위에 색상을 칠하게되면 캔버스 전체가 적용된 색상으로 적용

4.save 버튼 클릭시 캔버스의 그림이 저장

5.새로고침시 캔버스 clear
